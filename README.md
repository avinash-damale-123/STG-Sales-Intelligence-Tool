# STG Sales Intelligence Tool

## Project Documentation

The STG Sales Intelligence Tool is designed to provide insights and analysis to drive sales performance. It gathers data and generates reports to assist sales teams in making informed decisions.  

### Features:
- Data aggregation from multiple sources.
- Customizable reporting tools.
- Real-time analytics dashboard.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/avinash-damale-123/STG-Sales-Intelligence-Tool.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd STG-Sales-Intelligence-Tool
   ```

3. **Install required dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm start
   ```

5. **Access the dashboard in your browser** at `http://localhost:3000`
